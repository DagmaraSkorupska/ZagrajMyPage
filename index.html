Progressive Web App do wyszukiwania partnerów sportowych

Zobacz na GitHubie

📱 O projekcie

ZagrajMy to nowoczesna aplikacja webowa (PWA), która umożliwia użytkownikom wyszukiwanie partnerów sportowych na podstawie lokalizacji, poziomu zaawansowania i dostępnych terminów. Aplikacja integruje chat realtime oraz mapy – zbudowana jako szybkie MVP z potencjałem rozwoju do wersji natywnej.

🚀 Funkcje MVP

  • 🔐 Logowanie i rejestracja (Supabase Auth)
  • 👤 Tworzenie profilu i wybór sportów
  • 🧭 Lista dopasowań i widok mapy
  • 💬 Realtime chat z Supabase
  • 📲 Instalowalna PWA z offline cache

🛠️ Stack technologiczny

  • Frontend: Next.js (App Router), React, TypeScript, Tailwind CSS
  • Backend: Supabase (PostgreSQL, Auth, Realtime)
  • Inne: Zustand, Mapbox GL, Vercel, PWA
